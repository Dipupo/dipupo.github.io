Research

    Cars Assuring Resilient Security - CARS

      This research explores the security issues arising out of the growing use of electrical components that creates access points to vehicular distributed systems, that a potential hacker can exploit. Inherent security challenges such as distributed denial of service attacks, impersonation and fuzzy attacks are investigated to:
      • Establish theoretical foundations of fail-operational intrusion detection systems (FO-IDS)
      • Explore design space in the context of defending against attacks in automotive in-vehicle networks
      • Devise response and recovery protocols to ensure resilient operation, and
      • Evaluate the performance of the FO-IDS using actual vehicular test beds and data.

    Security and Data Provenance in Internet of Things (IoT)

      This project is aimed at developing a framework that can be used to securely collect data provenance on memory constrained embedded systems device.
