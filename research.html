Research

My research mainly focused on developing methods for ensuring the security and trustworthiness of embedded and distributed systems, including autonomous vehicles, Internet of Things (IoT), Internet of Vehicles (IoV), emerging networks, and smart home. My research involves the areas of computer networks, computer security, real-time systems, and cloud computing.

    Cars Assuring Resilient Security - CARS

      This research explores the security issues arising out of the growing use of electrical components that creates access points to vehicular distributed systems, that a potential hacker can exploit. Inherent security challenges such as distributed denial of service attacks, impersonation and fuzzy attacks are investigated to:
      • Establish theoretical foundations of fail-operational intrusion detection systems (FO-IDS)
      • Explore design space in the context of defending against attacks in automotive in-vehicle networks
      • Devise response and recovery protocols to ensure resilient operation, and
      • Evaluate the performance of the FO-IDS using actual vehicular test beds and data.

    Security and Data Provenance in Internet of Things (IoT)

      This project is aimed at developing a framework that can be used to securely collect data provenance on memory constrained embedded systems device. We seek to develope a provenance model for the IoT systems enabling:
      • The verification of systems correctness, integrity, and transparency of information shared across IoT platform
      • Providing mechanisms to ensure secure code and data provenance management on IoT platform
      • Investigate how provenance graphs can be used to detect anomalies in shared information on the IoT platform

    Security and Privacy in Emerging Network Technologies for CAVs.

      This project investigates approaches for providing security and privacy protections for named data networking (NDN) against data security threats for connected autonomous vehicles. The CAVs requires effective security of the integrated network frameworks and the cloud while providing strong user privacy. We aim to:
      • identify and assess security vulnerabilities within the network,
      • develop algorithms and techniques for content integrity and authorization and provide real-time guarantees on content dissemination in the NDN-enabled IoV network.

    Internet of Vehicles (IoV) Security in Smart City Ecosystem

      Technological advancements in sensor technology and communication techniques have brought the notion of IoV. Enhanced by the pervasiveness of the IoT technologies in transportation, automotive, and vehicular networks, the IoV is a promising model incorporating modern communication and connectivity technologies in wireless and mobile communications that permeate our lives. However, increased network access increases the attack surface, and securing the vehicles, passengers, and the environment is a looming challenge. This project explores and designs mechanisms for endpoints protection, trust and data integrity, data storage, and privacy in the IoV networks.
